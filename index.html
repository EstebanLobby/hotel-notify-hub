📋 Ejemplos de Variables

🏨 {hotel_name}

Descripción: Se reemplaza con el nombre completo del hotel

Ejemplo:

Texto: "Bienvenido a {hotel_name}"
Resultado: "Bienvenido a Demo Bitrix (Test Fabi)"
👤 {guest_name}

Descripción: Se reemplaza con el nombre del huésped

Ejemplo:

Texto: "Estimado {guest_name}"
Resultado: "Estimado Sr. Juan Pérez"
🎫 {reservation_number}

Descripción: Se reemplaza con el número único de reserva

Ejemplo:

Texto: "Reserva #{reservation_number}"
Resultado: "Reserva #jdgChd070000307dfbed-jhHERuhg454hg"
🔗 {checkin_url}

Descripción: Se reemplaza con la URL completa del formulario de check-in

Ejemplo:

Texto: "Complete el formulario: {checkin_url}"
Resultado: "Complete el formulario: https://selfcheckin.minihotel.info/?hotelID=..."

💡 Consejos de Uso

  • Obligatorias: {hotel_name} y {checkin_url} deben estar siempre presentes
  • Opcionales: {guest_name} y {reservation_number} son recomendadas para personalización
  • Formato: Siempre usa llaves {} alrededor del nombre de la variable
  • Mayúsculas: Respeta exactamente el nombre (hotel_name, no Hotel_Name)

📱 Vista Previa del Mensaje

📋 Variables utilizadas en este ejemplo:

  • {hotel_name} → Demo Bitrix (Test Fabi)
  • {guest_name} → Sr. Juan Pérez
  • {reservation_number} → jdgChd070000307dfbed-jhHERuhg454hg-gdfg070000307GRK
  • {checkin_url} → https://selfcheckin.minihotel.info/?hotelID=demobi05&reservationNumber=...
